# ECG InceptionTime Classification on PTB-XL

Репозиторий содержит реализацию модели InceptionTime для многометочной классификации электрокардиограмм по данным PTB-XL.

Работа выполнена в рамках учебной практики и направлена на исследование применения одномерных сверточных нейронных сетей для автоматического анализа ЭКГ-сигналов.

## Описание проекта

В проекте рассматривается задача классификации 12-канальных ЭКГ-записей по пяти диагностическим суперклассам PTB-XL:

- `NORM` — нормальная ЭКГ;
- `MI` — инфаркт миокарда;
- `STTC` — изменения сегмента ST и зубца T;
- `CD` — нарушения проводимости;
- `HYP` — гипертрофия.

Задача является многометочной, так как одна ЭКГ-запись может относиться сразу к нескольким диагностическим суперклассам.

## Используемый подход

В качестве основной модели используется ручная реализация архитектуры InceptionTime.

Модель применяет несколько параллельных одномерных сверток с разными размерами ядер. Такой подход позволяет анализировать ЭКГ-сигнал на нескольких временных масштабах и учитывать как короткие, так и более протяженные элементы кардиосигнала.

Основные особенности реализации:

- ручная реализация Inception-блоков;
- обработка 12-канальных ЭКГ-сигналов;
- использование функции потерь `BCEWithLogitsLoss` для многометочной классификации;
- учет дисбаланса классов через веса в функции потерь;
- подбор индивидуальных порогов классификации для каждого класса;
- оценка качества на validation и test folds PTB-XL;
- сохранение модели, таблиц метрик и графиков.
